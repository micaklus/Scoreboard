updateScore

fun updateScore(homeTeam: Team, awayTeam: Team, homeScore: Score, awayScore: Score, reasonForUpdate: Reason? = null)

Updates the score of an ongoing match between two teams.

Parameters

homeTeam

The home team for which the score will be updated.

awayTeam

The away team for which the score will be updated.

homeScore

The new score value for the home team.

awayScore

The new score value for the away team.

reasonForUpdate

A reason for updating the score. This is mandatory if the new score is lower than the current score.

Throws

IllegalArgumentException

If the match is not found or if the score is being lowered without a valid reason.
